Featured Project

AutomationInator

Enterprise BIM workflow orchestration · 2024 – Present · Bechtel

The Problem

Multi-site BIM projects require constant exports, format conversions, and health checks across half a dozen disconnected enterprise CAD platforms. The default workflow is manual, error-prone, license-hungry (every user needs every seat), and doesn't scale past a handful of sites.

What It Is

AutomationInator is a multi-software orchestration platform that runs the entire pipeline unattended on Windows service accounts. A manager clicks one button; exports, conversions, and copies run overnight across multiple VMs and report back when they're done.

7+ CAD platforms orchestrated
1 service-account license per VM (vs. per-user)
450 sites the architecture scales to (project-wide capacity)
25K+ lines of PowerShell (V1 + V2)

Highlights

  • V1 built solo in Notepad with pipe-delimited config files; deployed and running unattended at multiple Bechtel sites.
  • V2 is a full PowerShell + WinForms rewrite: dark-themed single-form UI, modular page architecture (Dashboard, Flows, Server Health, File Health, Setup, Options), self-locating paths.json config, structured JSON logging, live heartbeat monitoring.
  • Coordinates Tekla, Revit, SmartPlant 3D, Navisworks, AutoCAD, iConstruct, and PlantSight via service-account execution on multiple Windows VMs.
  • Solved real infrastructure problems along the way: DCOM configuration for service-account COM access, Session 0 process isolation, SPIOP/Intergraph Batch Manager license behavior, Roamer.exe batch invocation for Navisworks NWD exports, and SQL Server permissions for service accounts.
  • Designed for replication: drop in a new project, edit one JSON file, deploy.
